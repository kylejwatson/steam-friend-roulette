Steam Friend Roulette

What is this?

This is a little project I made in my time off work. It's a website to see which games you share with your friends; it could be useful to choose what to play together on a game night!

What tech is used?

The front end is an Angular web app using Material components. The back end is an express server, running multiple REST endpoints; this calls Steam's Web API to retrieve player and game detials, this data is then stored in cache.
